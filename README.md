# Python Learning Archive

Python을 공부하며 작성한 강의 실습, 알고리즘 풀이, 프로젝트 후보 코드를 모아둔 저장소입니다.  
일회성 테스트 파일과 로컬 실행 파일은 제거하고, 학습 흐름과 확장 가능성이 있는 주제 중심으로 정리합니다.

## Repository Map

| 경로 | 내용 |
| --- | --- |
| `codeTest/` | 백준, 프로그래머스, 온코더 문제 풀이 |
| `Programmers_Python/` | 프로그래머스 Python 강의 실습 |
| `Inflearn/` | 인프런 Python 강의, 알고리즘, 이미지 처리, 레시피 실습 |
| `Udemy_PythonBootcamp/` | Udemy Python Bootcamp 실습 |
| `crawling/` | BeautifulSoup, Selenium 기반 크롤링 실습 |
| `Django/` | Django 개념 정리와 실습 노트 |
| `REST_API/` | Flask 기반 REST API 실습 |
| `RPA/` | tkinter, PyQt, Outlook 자동화 실습 |
| `Streamlit/` | Streamlit 데이터 시각화 실습 |
| `MultiProcess&Thread/` | 멀티프로세스와 스레드 예제 |
| `STT_TTS/` | 음성 인식과 음성 합성 실습 |
| `translator/` | 번역 관련 실험 코드 |

## Project Candidates

추후 별도 프로젝트로 확장해볼 만한 주제입니다. 현재는 실험 코드에 가깝지만, 기능을 묶으면 독립 프로젝트로 가져갈 수 있습니다.

| 경로 | 확장 방향 |
| --- | --- |
| `STT_TTS/` | 음성 입력, 텍스트 변환, 음성 합성을 묶은 음성 처리 앱 |
| `RPA/` | GUI 자동화, 메일 자동화, PyQt 화면을 결합한 업무 자동화 도구 |
| `Streamlit/` | 공공데이터나 개인 데이터를 활용한 대시보드 |

## How To Run

대부분의 파일은 독립 실행형 예제입니다.

```bash
python path/to/example.py
```

## Notes

- 강의 실습과 문제 풀이가 함께 있는 학습용 저장소라 파일명이 강의명 또는 문제명 중심으로 되어 있습니다.
- `__pycache__`, `.DS_Store`, 가상환경, 로컬 실행 파일은 Git 추적 대상에서 제외합니다.
- 크롤링이나 음성 처리 예제에 필요한 드라이버와 실행 파일은 각자 환경에 맞게 설치해서 사용합니다.
- 단순 테스트, 샘플, 임시 메모 파일은 저장소에서 제거하고 필요한 내용만 주제별 디렉터리에 남깁니다.
